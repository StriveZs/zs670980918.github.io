# JAVA学前知识储备
## JAVA的三大版本
- JAVASE是用于桌面端的开发
- JAVAME适用于嵌入式小型设备的开发
- JAVAEE是WEB端的开发

## Hello World
新建一个文件，来编写hello world代码，注意文件名的后缀是.java结尾，而且文件名和类名要一样(区分大小写)。  

代码:

```
public class hello{
    public static void main(String [] args){
        System.out.print("Hello World");
    }
}
```
